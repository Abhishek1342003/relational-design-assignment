# Relational Design, Keys & Normalization

## Objective
This project converts raw CodeJudge CSV data into a normalized relational database schema.

The project includes:
- Schema understanding
- Entity identification
- Keys and constraints
- Normalization reasoning
- SQL schema creation
- ERD diagram
